Search Affinity

Affinity is a link tree visualization tool that takes a URL as an input and shows how the URL links to everything around it.

This tool uses a breadth first tree traversal algorithm to determine where the web scraper is pointed. The scraper itself loops over each a tag found in the DOM and pushes each href attribute onto an array. These arrays are sent to the client to be rendered and back into the algorithm to determine what is scraped next.

Clone Pen

Clone pen is a clone of the popular site codepen.io, it allows users to create self contained projects with HTML, CSS, and JavaScript.

Clone pen is a collaborative project from a team of four. I contributed in multiple areas including html, styling, and visual component creation, but spent most of my time implementing features and building the server and database.
Our site allows users to create self contained web pages from scratch. along with the ability to add external libraries and style-sheets dynamically. I couldn't have asked for a better team. Everyone worked hard and strove to push themselves.
