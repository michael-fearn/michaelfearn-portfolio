Search Affinity

Affinity is a link tree visualization tool that takes a URL as an input and shows how the URL link to everything around it.

This tool uses a breadth first tree traversal algorithm to determine where the web scraper is pointed. The scraper itself loops over each a tag found in the DOM and pushes each href attribute onto an array. These arrays are sent to the client to be rendered and back into the algorithm to determine what is scraped next.
